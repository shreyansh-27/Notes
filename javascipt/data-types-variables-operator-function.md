# Variables and Constants

```
let username = 'Max';
username = 'Meow';

const username = 'Max'
username = 'Meow' -> Not possible

```

Both let and const are data containers

# Variable Naming 
## Allowed
 > use camelCase
 >JS is case sensitive -> Ab is not equal to aB
 * Letters and digits are allowed
   * ageGroup5
 * starting with $ is allowed
   * $kindOfSpecial
 * starting with _ is allowed
   * _internalValue
## Not allowed
* let user_name -> allowed but not recommended
* let 21Players -> starting with digits not allowed
* let user-b -> No special characters!
* let let -> keywords not allowed
